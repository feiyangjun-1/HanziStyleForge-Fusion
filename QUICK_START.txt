HanziStyleForge Fusion 2.2 - Windows 11 quick start

1. Extract the project to a short local path, for example:
   C:\FontWork\HanziStyleForge-Fusion

2. Copy your fonts:
   fonts\target.ttf   (static TrueType glyf font; style source)
   refs\ref.otf       (static TrueType/CFF font; Han structure and coverage source)

3. Run:
   install_cuda130.bat

4. Verify the environment and project:
   verify_project.bat

5. Start or resume the complete workflow:
   run_months_resilient.bat

6. View read-only status:
   run_status.bat

7. Request a safe stop:
   request_safe_stop.bat
   The process exits after the next durable checkpoint. Run
   run_months_resilient.bat again to resume; the old request is cleared automatically.

8. Open the QA report when available:
   open_qa.bat

Final output:
   build\target-HanziStyleForge-Fusion.ttf

Core rules:
- Learn style only from target.ttf.
- Use ref.otf only for Han structure and output coverage.
- Rebuild every Han codepoint in the reference default Unicode cmap.
- Preserve and verify target non-Han glyphs and major OpenType data.
